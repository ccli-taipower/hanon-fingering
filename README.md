# 哈農鋼琴練習指南 · Hanon Piano Practice

**🎹 Live Demo：[ccli-taipower.github.io/hanon-fingering](https://ccli-taipower.github.io/hanon-fingering/)**

---

互動式哈農練習全方位工具，涵蓋音階指法查詢、練習類型說明、階段式練習計劃與進度追蹤。

## 功能

### 🎼 指法查詢
- 15 個大調 × 3 種調式（大調、和聲小調、旋律小調）= 45 種組合
- 右手上行／下行、左手上行／下行，共 4 種指法
- 拇指（藍色）與轉指點（綠色）高亮標示
- 拇指與轉指點下方直接顯示對應音名

### 📖 練習指南
- 8 種練習類型說明：音階、半音階、三和弦琶音、七和弦琶音、震音、八度、雙音、對稱式
- 8 個練習建議：節拍器三段法、連奏斷奏交替、轉指點慢動作、附點節奏變奏等

### 📋 練習計劃
- 5 個階段，每次 30 分鐘，按部就班
  - **S1 奠基期**：白鍵音階 ♩=60–84
  - **S2 擴展期**：小調與速度 ♩=84–100
  - **S3 技術期**：半音階與琶音 ♩=100–120
  - **S4 強化期**：八度與雙音 ♩=108–126
  - **S5 精熟期**：維持與表現 ♩=120+
- 每階段附查核點（勾選後顯示完成百分比）
- 勾選狀態自動儲存，下次開啟自動還原

### 📊 進度總覽
- 甘特圖：橫軸為練習次數（1–120 次），五個階段各占預估範圍
- 查核點完成度即時反映在甘特圖顏色深淺
- 統計目前進度、完成階段數、剩餘次數
