# 哈農音階指法查詢 · Hanon Scale Fingering Dashboard

互動式音階指法查詢工具，涵蓋所有大調及其平行小調（和聲小調、旋律小調），右手與左手、上行與下行。

**Live demo:** https://ccli-taipower.github.io/hanon-fingering/

## 功能

- 15 個大調（含等音調）× 3 種調式 = 45 種組合
- 右手上行 / 下行、左手上行 / 下行，共 4 種指法顯示
- 轉指點（拇指穿越）高亮標示
- 調式規律摘要（大調 / 和聲小調 / 旋律小調）

## 技術

純靜態 HTML + CSS + JavaScript，無任何依賴，直接以 GitHub Pages 部署。
